How does the bot work?

Makka Live Bot runs on your local machine using MakkaLauncher. This control center communicates with a backend bridge to securely start and stop the bot remotely.

What should I do first?
  1. Add the bot: Click the "Add Bot to Discord" link above to invite Makka to your server.
  2. Download the launcher: Get the MakkaLauncher.exe from the Config section above.
  3. Run the launcher: Execute the downloaded file on your host machine to connect it to this control center.
